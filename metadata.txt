description=Cria curvas de nivel no territorio brasileiro
version=1.0
author=Daniel Hulshof Saint Martin
email=daniel.hulshof@gmail.com

about=Este plugin cria curvas de nível a partir de dados geomorfométricos do território brasileiro obtidos no portal TOPODATA do INPE (http://www.dsr.inpe.br/topodata/). Os dados são baixados automaticamente e armazenados em cache local para reutilização. Funcionalidades: seleção de área de interesse, intervalo configurável entre curvas (1–1000 m), quatro níveis de suavização de terreno (Nenhum / Baixo / Médio / Alto), simbologia com curvas mestras e normais, rótulos de elevação e máscara de texto, reprojeção automática para o SRC do projeto, suporte a proxy com autenticação. ATENÇÃO: este plugin cobre apenas o território brasileiro. Para cobertura mundial, utilize o plugin Contour Lines: https://github.com/DanielHSMartin/ContourLines

tracker=https://github.com/DanielHSMartin/CurvaDeNivel/issues
repository=https://github.com/DanielHSMartin/CurvaDeNivel
